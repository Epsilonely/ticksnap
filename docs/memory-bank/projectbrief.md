# Project Brief - TickSnap

## Project Overview

**Project Name:** TickSnap  
**Type:** Desktop cryptocurrency trading application  
**Platform:** Electron-based cross-platform desktop app  
**Primary Exchanges:** Binance, Upbit

## Core Requirements

### Essential Features

1. Real-time cryptocurrency price monitoring
2. Multi-exchange support (Binance & Upbit)
3. Portfolio management and tracking
4. Trading functionality with exchange APIs
5. Favorites/watchlist system
6. Leaderboard for top performers
7. Detailed coin information display

### Technical Requirements

- Desktop application using Electron
- Real-time data updates via WebSocket
- Secure API key management
- Responsive UI with modern design
- TypeScript for type safety
- State management for complex data flows

## Project Goals

1. **User Experience:** Provide a unified interface for monitoring multiple exchanges
2. **Performance:** Real-time updates with minimal latency
3. **Security:** Safe handling of API credentials and trading operations
4. **Reliability:** Stable connection management and error handling
5. **Extensibility:** Architecture that supports adding more exchanges

## Success Criteria

- Successful real-time price updates from both exchanges
- Functional trading operations through exchange APIs
- Accurate portfolio tracking and calculations
- Stable WebSocket connections with automatic reconnection
- Clean, intuitive user interface

## Constraints

- Must work as standalone desktop application
- Must support Windows (primary), with potential for cross-platform
- Must handle API rate limits appropriately
- Must secure user credentials locally
